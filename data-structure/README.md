### 5장. 연결 리스트 Linked List

- 연결 리스트

  리스트의 시작 원소(head) 부터 순회를 시작한다.

  node의 next 프로퍼티에 다음 값의 참조를 갖는다. (사실상 중첩 객체인 듯..)

- 이중 연결 리스트

  node에 next 뿐만 아니라 prev의 프로퍼티를 추가한다. 즉, 이전 값을 갖게 되어 리스트의 마지막(tail) 부터 순회가 가능하다.
