테스트



```javascript
describe('pow', function() {
    it('주어진 숫자의 n 제곱', function(){
        assert.equal(pow(2,3), 8);
    })
})
```



### describe

- 구현하고자 하는 기능에 대한 명세

### it

- 첫 번재 인수에는 특정 유스케이스에 대한 설명(자연어)
- 두 번째 인수에는 유스케이스에 대한 테스트 함수
